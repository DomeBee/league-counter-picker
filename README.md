# league-counter-picker
A simple Python tool that recommends the best counter pick based on the enemy champion. It uses a scoring system to evaluate matchups and provides reasons and tips for each recommendation.

My Aim:

-Improve Python fundamentals through a real project
-Practice working with structured data 
-Build logic to evaluate and compare multiple options
-Gradually refactor code to be cleaner and more modular
-Turn a simple script into a more complete tool over time

v 1.0- Initial Version:
-Basic counter selection logic
-User input handling
-Displays all counters and best pick
